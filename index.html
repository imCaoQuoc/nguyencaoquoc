FAQ

Can I host this on GitHub Pages?

Yes. Commit this index.html to the root of your gh-pages branch (or set Pages to use the root/main) and enable GitHub Pages in repo settings.

How do I customize colors and spacing?

Edit the CSS variables in :root and [data-theme="light"] at the top of this file.

Is JavaScript required?

Only for theme persistence and a small demo that injects the feature cards. You can make it 100% static by inlining those cards in HTML.
